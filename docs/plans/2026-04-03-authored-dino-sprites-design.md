# Authored Dinosaur Sprites — Design

## Context
The game currently has 9 dinosaurs rendered procedurally (basic geometric shapes) except for Brachiosaurus which has hand-crafted pixel-art sprites generated by `scripts/generate_authored_assets.js`. The Brachiosaurus looks significantly better. We want to bring all dinosaurs up to that quality level.

## Approach: Proof of Concept with T-Rex
Build authored assets for Tyrannosaurus first to validate that pixel-art sprites look good on top of the existing procedural biome backgrounds. If it works, extend to all 8 remaining dinos.

## Per-Dinosaur Deliverables
1. **Sprite strip PNG** — 8 frames at 72×96px each (576×96px total), 6fps
2. **Contact preview PNG** — 210×120px silhouette for tracking panel
3. **Guide preview PNG** — 220×132px colored preview for field guide

## T-Rex Sprite Design
- Massive skull with visible jaw/teeth, tiny forearms, thick powerful hind legs, heavy counterbalancing tail
- Color palette: deep reds and browns with darker accent shading
- 8 animation frames: jaw open/close cycle, subtle body bob, tail sway, leg shift
- Built using existing pixel primitives from the Brachiosaurus code (`drawTube`, `drawEllipsoid`, `setPixel`)

## Key Architecture Change: Hybrid Rendering
Currently authored sprites only work inside `AUTHORED_SCENES` (which includes a full custom background). We need a hybrid mode:

1. `AUTHORED_DINOSAURS` gets a new `sprite` field with default placement config
2. When a visible dino has authored sprite data but no authored scene:
   - Procedural background renders normally on canvas
   - Authored sprite element is positioned over the canvas at computed coordinates
   - Sprite strip animates as usual
3. When dino is NOT visible, sprite element hides (existing behavior)

## Files to Modify
- `scripts/generate_authored_assets.js` — Add T-Rex drawing functions
- `app.js` — Extend `AUTHORED_DINOSAURS`, add hybrid sprite rendering in `renderAuthoredScene()` / `showEncounter()`
- `assets/authored/tyrannosaurus/` — Generated output PNGs

## Success Criteria
- T-Rex pixel art looks natural on procedural biome backgrounds
- Animation is smooth at 6fps
- Contact and guide previews render correctly
- No visual glitches during encounter/photo/exit flows

## Future: Remaining 8 Dinos
If proof of concept succeeds, extend to: Coelophysis, Plateosaurus, Postosuchus, Stegosaurus, Allosaurus, Parasaurolophus, Triceratops. Each follows the same pipeline.
